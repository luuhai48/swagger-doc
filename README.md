# Install

Install packages:

```bash
npm install
```

Or

```bash
yarn
```

# Usage

### Edit document

```bash
npm run edit
```

Or

```bash
yarn edit
```

The web server will be started. Follow the link in the terminal to edit the swagger document.

### View document

Open `index.html` file using `live-server` on vscode.

Or

```bash
npm run open
```

Or

```bash
yarn open
```
